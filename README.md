## üìå Project Overview

This project is a **Workout Tracker** application built using the **MERN stack** (MongoDB, Express, React, Node.js).  
The application allows users to log workouts, add multiple sets (reps + weight), write notes, and store data in the cloud using MongoDB Atlas.

This project demonstrates:
- Backend API development with Express
- Data modeling and persistence using MongoDB + Mongoose
- Frontend UI development with React (Vite)
- Connecting client and server applications
- Handling CORS and environment configuration
- Debugging full-stack integration issues
